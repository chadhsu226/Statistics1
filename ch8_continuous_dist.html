Chapter 8　連續機率分配 (Continuous Probability Distributions)

第 7 章的隨機變數用「數」的；本章的隨機變數用「量」的——時間、重量、金額可以取區間內的任何值。連續世界的機率規則改變了：單點機率為零，機率變成「曲線下的面積」。本章的主角是統計學最重要的分配——常態分配：學會標準化與查表，你就拿到了第 9、10 章統計推論的鑰匙。

§8.1　機率密度函數與均勻分配 (Probability Density Functions)

一、連續隨機變數的機率規則

📐 機率密度函數 (PDF)

連續隨機變數以機率密度函數 $f(x)$ 描述，必須滿足：

  1. 非負：對所有 $x$，$f(x) \geq 0$。
  2. 總面積為 1：$f(x)$ 曲線下的總面積等於 1。

機率 ＝ 曲線下的面積：$P(x_1 < X < x_2)$ ＝ 曲線下介於 $x_1$ 與 $x_2$ 之間的面積。

單點機率為零：$P(X = x) = 0$（單點的「面積」寬度為 0）。因此對連續變數而言，$P(X < a)$ 與 $P(X \leq a)$ 相等——「＜」與「≤」不必計較。這點與離散變數完全不同！

二、均勻分配 (Uniform Distribution)

📐 均勻分配

$$f(x) = \frac{1}{b-a}, \quad a \leq x \leq b \qquad\qquad E(X) = \frac{a+b}{2}$$

密度是水平直線（矩形分配），區間內每一段等寬範圍的機率都相同。機率計算就是矩形面積：

$$P(x_1 < X < x_2) = (x_2 - x_1) \times \frac{1}{b-a}$$

📝 範例 8.1　捷運月台的等候時間

情境：離峰時段捷運每 12 分鐘一班。小葉不看時刻表、隨機抵達月台，因此她的等候時間 $X$ 服從 $U(0, 12)$ 的均勻分配。

a. 機率密度函數：$f(x) = \dfrac{1}{12}$，$0 \leq x \leq 12$；平均等候 $E(X) = \dfrac{0+12}{2} = 6$ 分鐘。

b. 等超過 8 分鐘的機率：

$$P(X > 8) = (12 - 8) \times \frac{1}{12} = \frac{4}{12} = 0.333$$

c. 等 2 到 5 分鐘的機率：

$$P(2 < X < 5) = (5 - 2) \times \frac{1}{12} = 0.25$$

d. 恰好等 6 分鐘的機率：$P(X = 6) = 0$（連續變數單點機率為零）。

✅ 均勻分配是「完全無知」的模型——除了範圍之外，對 $X$ 落在哪裡毫無線索。多數真實變數（如身高、誤差）反而會向中央集中，這就輪到常態分配登場。

§8.2　常態分配 (Normal Distribution)

一、常態分配的性質

📐 常態分配 $X \sim N(\mu, \sigma^2)$

  • 由兩個參數決定：平均數 $\mu$（位置——曲線的中心）與標準差 $\sigma$（形狀——曲線的胖瘦）。
  • 圖形為對稱的鐘形：平均數＝中位數＝眾數，皆位於 $\mu$。
  • 兩尾向左右無限延伸、逐漸貼近橫軸但永不觸及。
  • 第 4 章的經驗法則（68–95–99.7）正是常態分配的精確性質。

二、標準化與查表（正查）

📐 標準常態分配與 z 轉換

把 $\mu = 0$、$\sigma = 1$ 的常態分配稱為標準常態分配 $Z$。任何常態變數都能透過標準化轉換成 $Z$：

$$Z = \frac{X - \mu}{\sigma}$$

$z$ 值的意義：「$x$ 距離平均數幾個標準差」。z 表（見互動查表系統）給出累積機率 $P(Z < z)$，即曲線下 $z$ 左邊的面積。三個常用轉換：
右尾 $P(Z > z) = 1 - P(Z < z)$；區間 $P(a < Z < b) = P(Z < b) - P(Z < a)$；對稱 $P(Z < -z) = P(Z > z)$。

Z 轉換的幾何意義：平移＋伸縮

標準化公式 $z = (x - \mu)/\sigma$ 其實是兩個幾何動作的組合：

  1. 平移 (Shift)：分子的「$-\,\mu$」把整個鐘形往左移 $\mu$ 單位，中心從 $\mu$ 移到 0——邊界跟著移動：$x \to x - \mu$。
  2. 伸縮 (Scale)：分母的「$\div\,\sigma$」把橫軸壓縮（或拉伸）$\sigma$ 倍，胖瘦變成 1——邊界跟著縮放：$x - \mu \to (x-\mu)/\sigma$。

整個過程中，著色區域的面積（機率）始終不變——這就是「任何常態分配的機率都能改查 Z 表」的原因。用下面的動畫親眼看一次：

預設 $X \sim N(5,\ 2^2)$、求 $P(X > 6)$。按「▶ 播放」看兩步驟連續變形；也可逐步點按，或修改 μ、σ、邊界後重新播放。灰色虛線是目標——標準常態 $N(0,1)$。

📝 主範例 8.2　外送時間的三種機率

情境：某外送平台的送達時間 $X$ 近似常態分配，$\mu = 32$ 分鐘、$\sigma = 5$ 分鐘。

a. 40 分鐘內送達的機率 $P(X < 40)$

$$z = \frac{40 - 32}{5} = 1.6 \qquad P(X < 40) = P(Z < 1.6) = 0.9452$$

b. 超過 30 分鐘的機率 $P(X > 30)$

$$z = \frac{30 - 32}{5} = -0.4 \qquad P(X > 30) = P(Z > -0.4) = 1 - 0.3446 = 0.6554$$

c. 25 到 35 分鐘之間的機率 $P(25 < X < 35)$

$$z_1 = \frac{25-32}{5} = -1.4 \qquad z_2 = \frac{35-32}{5} = 0.6$$
$$P(-1.4 < Z < 0.6) = P(Z < 0.6) - P(Z < -1.4) = 0.7257 - 0.0808 = 0.6449$$
✅ 查表三步驟：標準化 → 畫圖著色 → 組合面積。強烈建議每題都先畫一張小草圖，把要求的面積塗出來——右尾要用 1 減、區間要相減，畫了圖就不會弄反。

三、反向查表與反標準化

📐 分位數記號 $z_A$ 與反標準化

$z_A$：右尾面積為 $A$ 的 z 值，即 $P(Z > z_A) = A$。查表方式：在表身找累積機率 $1 - A$，讀出對應的 $z$。

$$z_{0.10} = 1.28 \qquad z_{0.05} = 1.645 \qquad z_{0.025} = 1.96 \qquad z_{0.01} = 2.33 \qquad z_{0.005} = 2.575$$

反標準化：求出 $z$ 後換回原始尺度：

$$x = \mu + z\,\sigma$$

這五個 $z_A$ 值在第 10 章以後會不斷出現（信賴區間、假設檢定的臨界值），值得現在就記熟。

📝 主範例 8.3　白金會員的門檻

情境：某連鎖書店會員的年消費金額近似常態分配，$\mu = 24{,}000$ 元、$\sigma = 6{,}000$ 元。行銷部門想把消費最高的前 10% 會員升級為白金會員。年消費門檻應設在多少？

Step 1　找 z 值：前 10% ＝ 右尾面積 $A = 0.10$ → $z_{0.10} = 1.28$。

Step 2　反標準化：

$$x = \mu + z\,\sigma = 24{,}000 + 1.28 \times 6{,}000 = 31{,}680$$
✅ 門檻約 $31,680 元（實務上取整為 32,000）。注意方向：「前 10%」在右尾，$z$ 為正；若是「最低的 10%」則 $z = -1.28$、門檻為 $24{,}000 - 7{,}680 = 16{,}320$ 元。

📝 範例 8.4　保固期該設幾個月？

情境：某品牌電動機車電池的壽命近似常態分配，$\mu = 36$ 個月、$\sigma = 4$ 個月。公司希望「保固期內故障而需免費更換」的比例不超過 2.5%。保固期最長可以設幾個月？

Step 1：故障比例 2.5% 在左尾 → $z = -z_{0.025} = -1.96$。

Step 2：

$$x = \mu + z\,\sigma = 36 + (-1.96)(4) = 36 - 7.84 = 28.16$$
✅ 保固期設 28 個月，理論更換率約 2.3%（略低於 2.5% 的上限，因為 28 比 28.16 更嚴格）。這類「由容許風險反推規格」的計算，是常態分配在商業上最實用的形態之一。

§8.3　指數分配 選讀補充

📐 指數分配 (Exponential Distribution)

描述「事件與事件之間的間隔時間」。若某事件的發生次數服從卜瓦松分配（第 7 章，平均每單位時間 $\lambda$ 次），則相鄰兩次事件的間隔時間 $X$ 服從指數分配：

$$f(x) = \lambda e^{-\lambda x},\ x \geq 0 \qquad P(X > x) = e^{-\lambda x} \qquad E(X) = \frac{1}{\lambda}$$

右偏、從 $\lambda$ 高度起跳向右衰減。典型場景：來電間隔、機台故障間隔、顧客到店間隔。

📝 範例 8.5　下一通電話多久後進來？（延續範例 7.5）

情境：客服深夜來電服從卜瓦松分配、平均每小時 $\lambda = 3$ 通。剛掛掉一通電話，「接下來 20 分鐘（$1/3$ 小時）都沒有來電」的機率是多少？
$$P(X > \tfrac{1}{3}) = e^{-3 \times 1/3} = e^{-1} = 0.368$$
✅ 平均間隔 $E(X) = 1/3$ 小時 ＝ 20 分鐘，但「超過平均間隔」的機率只有 36.8%（不是 50%）——指數分配右偏，多數間隔比平均短、少數間隔特別長。卜瓦松（數次數）與指數（量間隔）是同一個隨機過程的兩面。

📚 綜合練習 (Comprehensive Practice)

以下習題涵蓋均勻分配、常態正查與反查，請先獨立作答，再展開解答確認。（z 值可利用互動查表系統）

I. 概念複習

  1. 為什麼連續隨機變數的單點機率 $P(X = x) = 0$？這是否代表「$X = x$ 不可能發生」？

    ▶ 點此展開解答

    機率＝曲線下面積，單點的寬度為 0，面積必為 0。

    但機率為零不等於不可能——$X$ 總會落在某個確切的值上。正確理解：連續變數的機率只對「區間」有意義，「恰好等於某值」的機率在測度上為零。實務中我們問的也總是區間問題（「40 分鐘內」而非「恰好 40.000... 分鐘」）。

  2. 常態分配的 $\mu$ 與 $\sigma$ 分別控制曲線的什麼？兩個常態分配 $N(50, 5^2)$ 與 $N(50, 10^2)$ 的圖形差異為何？

    ▶ 點此展開解答

    $\mu$ 控制位置（中心在哪）；$\sigma$ 控制形狀（胖瘦）。

    兩者中心相同（都在 50），但 $N(50, 10^2)$ 更矮更寬（資料更分散），$N(50, 5^2)$ 更高更窄。總面積都是 1，所以變寬必然變矮。

  3. 標準化（z 轉換）的目的是什麼？為什麼一張 z 表就能處理所有的常態分配問題？

    ▶ 點此展開解答

    標準化把任意 $N(\mu, \sigma^2)$ 線性轉換為同一個標準常態 $N(0,1)$，且轉換保持面積（機率）不變。

    因此無論原始單位是分鐘、元或毫升，「距離平均幾個標準差」的機率結構完全相同——一張 $Z$ 表通吃所有常態問題。

II. 計算與應用

  1. 8.1　自動洗車機的服務時間

    某自動洗車機的單次洗車時間 $X$（分鐘）服從 $U(4, 10)$。

    • a. 寫出 $f(x)$ 並求平均服務時間。
    • b. 求 $P(X < 6)$。
    • c. 求 $P(5 < X < 8)$。
    ▶ 點此展開解答

    a. $f(x) = \dfrac{1}{10-4} = \dfrac{1}{6}$，$4 \leq x \leq 10$；$E(X) = (4+10)/2 = 7$ 分鐘。

    b. $P(X < 6) = (6-4) \times \dfrac{1}{6} = \dfrac{2}{6} = 0.333$

    c. $P(5 < X < 8) = (8-5) \times \dfrac{1}{6} = 0.5$

  2. 8.2　z 表基本功

    $X \sim N(50, 8^2)$。求：

    • a. $P(X < 62)$
    • b. $P(X > 46)$
    • c. $P(42 < X < 58)$
    ▶ 點此展開解答

    a. $z = (62-50)/8 = 1.5$；$P(Z < 1.5) = 0.9332$

    b. $z = (46-50)/8 = -0.5$；$P(Z > -0.5) = 1 - 0.3085 = 0.6915$

    c. $z_1 = -1$、$z_2 = 1$；$P(-1 < Z < 1) = 0.8413 - 0.1587 = 0.6827$（正是經驗法則的 68%！）

  3. 8.3　高鐵通勤的時間管理

    阿哲每天通勤（門到門）時間 $\sim N(90, 6^2)$ 分鐘。

    • a. 他提早 100 分鐘出門，準時抵達的機率？
    • b. 某天他只剩 78 分鐘可用，遲到的機率？
    • c. 若想有 99% 的把握準時，應該提早多久出門？
    ▶ 點此展開解答

    a. $z = (100-90)/6 = 1.67$；$P(X < 100) = 0.9525$。

    b. $z = (78-90)/6 = -2$；$P(X > 78) = 1 - 0.0228 = 0.9772$ → 遲到機率 $= P(X > 78)$？注意題意：可用時間 78 分鐘，「通勤時間超過 78 分鐘」就會遲到，故遲到機率 $= 0.9772$——幾乎肯定遲到。

    c. 99% 把握 → 左尾累積 0.99 → $z_{0.01} = 2.33$；$x = 90 + 2.33 \times 6 = 104$ 分鐘。提早約 104 分鐘出門。

  4. 8.4　反查 z 值練習
    • a. 求 $z_{0.005}$（右尾 0.5%）。
    • b. 求使中央 98% 的面積落在 $\pm z$ 之間的 $z$ 值。
    • c. 求 $P(Z < z) = 0.20$ 的 $z$ 值。
    ▶ 點此展開解答

    a. 累積機率 $= 0.995$ → $z_{0.005} = 2.575$（表中 2.57 與 2.58 之間取中）。

    b. 兩尾各 1% → $z_{0.01} = 2.33$，即 $\pm 2.33$。

    c. 左尾 20% 在平均數左側 → 由對稱性，$P(Z < z)=0.20$ 對應 $z = -0.84$（因 $P(Z < 0.84) \approx 0.80$）。

  5. 8.5　行動電源的續航宣傳

    某行動電源的實際續航力 $\sim N(30, 2.5^2)$ 小時。行銷部門要在廣告上寫「續航 ○○ 小時以上」，並希望至少 90% 的產品能達到宣稱值。宣稱值最高可以寫多少？

    ▶ 點此展開解答

    「90% 的產品超過宣稱值 $x$」→ $P(X > x) = 0.90$ → $x$ 在左尾、左尾面積 10% → $z = -1.28$。

    $$x = 30 + (-1.28)(2.5) = 30 - 3.2 = 26.8$$

    宣稱「26 小時以上」（取整數並更保守）。若貪心寫 30 小時，會有一半的產品「不符廣告」——這就是為什麼規格標示通常明顯低於平均值。

  6. 8.6　咖啡機的填充量設定

    自動咖啡機每杯的填充量標準差 $\sigma = 6$ mL（機器固有精度），平均填充量 $\mu$ 可由店家設定。杯子容量 360 mL，店長希望溢出的機率不超過 1%。$\mu$ 最高可設定為多少？

    ▶ 點此展開解答

    溢出 ⇔ $X > 360$；要求 $P(X > 360) \leq 0.01$ → $\dfrac{360 - \mu}{6} \geq z_{0.01} = 2.33$。

    $$\mu \leq 360 - 2.33 \times 6 = 360 - 13.98 = 346.0 \text{ mL}$$

    最高設 346 mL。想多裝一點（提高 $\mu$）就得先讓機器更精準（降低 $\sigma$）——品質工程「規格 = 平均 + 安全邊際」的基本邏輯。

📌 Chapter Summary

  • 連續變數：機率＝密度曲線下的面積；$f(x) \geq 0$、總面積 1；單點機率為零，故 $<$ 與 $\leq$ 等價。
  • 均勻分配 $U(a,b)$：$f(x) = 1/(b-a)$，機率＝矩形面積；$E(X) = (a+b)/2$。
  • 常態分配 $N(\mu, \sigma^2)$：對稱鐘形；$\mu$ 管位置、$\sigma$ 管胖瘦；經驗法則是其精確性質。
  • 正查：標準化 $z = (x-\mu)/\sigma$ → 畫圖著色 → 查表組合（右尾用 $1-$、區間用相減）。
  • 反查：由機率找 $z_A$（$z_{0.10}=1.28$、$z_{0.05}=1.645$、$z_{0.025}=1.96$、$z_{0.01}=2.33$、$z_{0.005}=2.575$），再反標準化 $x = \mu + z\sigma$；先確認方向（左尾／右尾／中央）。
  • 指數分配（選讀）：事件間隔時間，$P(X > x) = e^{-\lambda x}$，與卜瓦松是同一過程的兩面。
