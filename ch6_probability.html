Chapter 6　機率 (Probability)

第 1 章說過：統計推論的結論「有特定機率是正確的」。要讓這句話有意義，我們需要一套量化不確定性的數學語言——機率論。本章建立機率的基本詞彙（樣本空間、事件）、三種機率類型（聯合、邊際、條件）、三大運算法則（補集、加法、乘法），並以貝氏定理作壓軸：如何用新證據修正原有的判斷。

🎯 Learning Objectives

  • 理解隨機試驗、樣本空間、事件的定義，會用三種方法指派機率。
  • 能從交叉分類表計算聯合機率、邊際機率、條件機率。
  • 能判斷兩事件是否獨立、是否互斥，並理解兩者的差異。
  • 能運用補集、加法、乘法法則與樹狀圖計算複合事件的機率。
  • 能以貝氏定理計算後驗機率，並理解「篩檢陽性 ≠ 幾乎確定有病」的直覺翻轉。

§6.1　機率的基本概念 (Basic Concepts of Probability)

一、核心術語

📐 三個基本詞彙

  • 隨機試驗 (Random Experiment)：結果事先無法確定的過程。例：投擲骰子、觀察明天某飲料店的來客數。
  • 樣本空間 (Sample Space, $S$)：隨機試驗所有可能結果的集合。其中的每個結果稱為簡單事件，必須互斥且周延（不重複、不遺漏）。
  • 事件 (Event, $A$)：樣本空間的子集合——我們感興趣的一個或多個結果。

機率的兩條天條：① 每個結果的機率介於 0 與 1 之間：$0 \leq P(O_i) \leq 1$；② 所有結果的機率總和為 1：$\sum P(O_i) = 1$。任何違反這兩條的「機率」都是錯的。

二、指派機率的三種方法

方法做法適用情境
古典方法 結果機會均等時，$P(A) = \dfrac{A \text{ 包含的結果數}}{S \text{ 的總結果數}}$ 骰子、撲克牌、抽籤等對稱機制
相對頻率方法 以大量重複觀察中事件發生的比例估計機率 商業數據：退貨率、點擊率、良率
主觀方法 依據知識、經驗做出的合理判斷 無法重複的事件：新產品成功機率、球賽勝率

📝 範例 6.1　三種方法各就各位

情境：判斷下列機率各是用哪種方法指派的：
a. 夜市戳戳樂共 40 格，其中 5 格有大獎，戳中大獎的機率是 $5/40 = 0.125$。
b. 某電商過去一年 80,000 筆訂單中有 2,400 筆退貨，故估計退貨機率為 0.03。
c. 創投合夥人評估某新創公司三年內成功上市的機率約為 0.15。

解答：a. 古典方法（40 格機會均等）；b. 相對頻率方法（大量歷史資料的比例）；c. 主觀方法（一次性事件，靠專業判斷）。

✅ 三種方法指派出的機率，後續的運算規則（§6.2–6.4）完全相同——機率的來源可以不同，數學是同一套。

§6.2　聯合、邊際與條件機率 (Joint, Marginal, and Conditional Probability)

實務上最常見的機率問題涉及兩個事件的組合。先定義事件之間的關係：

📐 事件的關係與三種機率

  • 交集 $A \cap B$（「且」）：$A$、$B$ 同時發生 → 其機率稱為聯合機率 $P(A \cap B)$。
  • 聯集 $A \cup B$（「或」）：$A$ 發生、$B$ 發生、或同時發生。
  • 補集 $A^C$：$A$ 不發生。
  • 邊際機率 $P(A)$：單一事件發生的機率——在聯合機率表中，把對應的整列（或整行）加總。
  • 條件機率：已知 $B$ 發生的條件下，$A$ 發生的機率：
    $$P(A \mid B) = \frac{P(A \cap B)}{P(B)}$$

條件機率的直覺：「已知 $B$ 發生」等於把世界縮小到 $B$ 這個圈圈裡，再問 $A$ 在圈圈裡占多大比例。

📝 主範例 6.2　線上課程平台：完課與加購

情境：某線上課程平台分析大量學員紀錄，整理出「是否完課」與「是否加購進階課程」的聯合機率表。管理層想知道：完課行為與加購行為有沒有關係？
加購進階課 $U$未加購 $U^C$邊際機率
完課 $C$0.180.220.40
未完課 $C^C$0.060.540.60
邊際機率0.240.761.00

a. 邊際機率：$P(C) = 0.18 + 0.22 = 0.40$（四成學員完課）；$P(U) = 0.18 + 0.06 = 0.24$（24% 學員加購）。

b. 條件機率——完課的學員中，加購的比例：

$$P(U \mid C) = \frac{P(U \cap C)}{P(C)} = \frac{0.18}{0.40} = 0.45$$

未完課的學員中：$P(U \mid C^C) = 0.06 / 0.60 = 0.10$。

c. 判讀：完課學員的加購率（45%）是未完課者（10%）的 4.5 倍——兩個行為高度相關。

✅ 決策意涵：提高完課率（提醒推播、學習小組）可能連帶提升營收。但注意——這是觀察資料，「完課→更願意買」與「本來就積極的人既完課又加購」兩種解釋都成立（第 5 章的老朋友：相關 ≠ 因果）。

獨立事件 (Independent Events)

📐 獨立的定義與檢驗

事件 $A$ 與 $B$ 獨立，若且唯若：

$$P(A \mid B) = P(A) \qquad \Leftrightarrow \qquad P(A \cap B) = P(A) \times P(B)$$

白話：「知道 $B$ 發生了」完全不改變你對 $A$ 的評估。檢驗方式：比較 $P(A \mid B)$ 與 $P(A)$ 是否相等（或聯合機率是否等於邊際機率的乘積）。

以主範例 6.2 檢驗：$P(U \mid C) = 0.45 \neq P(U) = 0.24$ → 完課與加購不獨立（相關）。

⚠️ 獨立 vs 互斥——最容易搞混的一對：
互斥 (mutually exclusive)：$A$、$B$ 不能同時發生，$P(A \cap B) = 0$。
獨立 (independent)：$A$ 發生與否不影響 $B$ 的機率。
兩者不但不同，而且（當 $P(A), P(B) > 0$ 時）互斥的事件必然不獨立——因為知道 $A$ 發生，就確定 $B$ 沒發生，資訊影響巨大。

§6.3　機率法則與樹狀圖 (Probability Rules and Trees)

📐 三大法則

  • 補集法則：$P(A^C) = 1 - P(A)$

    最重要的應用：「至少一次」問題——$P(\text{至少一次}) = 1 - P(\text{一次都沒有})$。

  • 加法法則：$P(A \cup B) = P(A) + P(B) - P(A \cap B)$

    減去交集是為了避免重複計算；互斥時交集為 0，可直接相加。

  • 乘法法則：$P(A \cap B) = P(A) \times P(B \mid A)$

    獨立時簡化為 $P(A) \times P(B)$。這其實就是條件機率公式移項的結果。

📝 範例 6.3　加法法則：手搖飲的集點客群

情境：「沐茶」的會員資料顯示：45% 的會員本月買過茶飲（$T$）、30% 買過咖啡（$F$）、15% 兩者都買過。隨機抽一位會員，本月至少買過其中一類的機率是多少？完全沒買的機率呢？
$$P(T \cup F) = 0.45 + 0.30 - 0.15 = 0.60$$
$$P(\text{都沒買}) = 1 - P(T \cup F) = 1 - 0.60 = 0.40$$
✅ 若直接 0.45 + 0.30 = 0.75，會把「兩者都買」的 15% 算兩次。畫文氏圖確認：只買茶 0.30、只買咖啡 0.15、都買 0.15、都沒買 0.40，總和 = 1 ✓。

📝 主範例 6.4　樹狀圖：外送平台的遲到分析

情境：某外送平台的訂單中，70% 來自市中心（$D$）、30% 來自郊區（$D^C$）。歷史資料顯示：市中心訂單的遲到率為 10%，郊區訂單的遲到率為 30%。隨機抽一筆訂單，遲到（$L$）的機率是多少？

樹狀圖的規則：第一層分支放邊際機率，第二層放條件機率；沿路徑相乘得到終點的聯合機率；四條路徑的機率總和必為 1。

$$P(L) = P(D)P(L \mid D) + P(D^C)P(L \mid D^C) = (0.7)(0.1) + (0.3)(0.3) = 0.07 + 0.09 = 0.16$$
✅ 這條「把各分支的貢獻加總」的公式稱為全機率法則：整體遲到率 16%，其中市中心貢獻 7 個百分點、郊區貢獻 9 個百分點——郊區訂單雖只占三成，卻貢獻了超過一半的遲到。

📝 範例 6.5　「至少一次」的補集技巧

情境：某手遊的抽卡機制中，每次抽卡獲得稀有角色的機率為 5%，且每次抽卡獨立。小安打算抽 10 次，「至少抽中一次稀有角色」的機率是多少？

直接計算「至少一次」要窮舉抽中 1 次、2 次⋯⋯10 次，非常繁瑣。改用補集：

$$P(\text{至少一次}) = 1 - P(\text{十次全落空}) = 1 - (0.95)^{10} = 1 - 0.5987 = 0.4013$$
✅ 十連抽的中獎機率其實只有四成——不是很多人直覺的「5% × 10 = 50%」。（順帶一提：機率的直覺常常出錯，這正是要學機率論的原因。）

§6.4　貝氏定理 (Bayes' Theorem)

樹狀圖是「順著時間算」：先知道訂單來源，再算遲到機率。貝氏定理則是「倒著推」：已經觀察到結果（遲到了），回推原因（來自哪一區）的機率。

📐 貝氏定理

$$P(A \mid E) = \frac{P(A)\,P(E \mid A)}{P(E)} = \frac{P(A)\,P(E \mid A)}{P(A)\,P(E \mid A) + P(A^C)\,P(E \mid A^C)}$$
  • $P(A)$：先驗機率——看到證據之前對 $A$ 的評估。
  • $P(E \mid A)$：似然——若 $A$ 為真，出現這個證據的機率。
  • $P(A \mid E)$：後驗機率——看到證據之後，修正過的評估。

分母就是全機率法則算出的 $P(E)$。實務操作：先畫樹狀圖算出所有聯合機率，後驗機率 ＝ 目標路徑 ÷ 所有含證據 $E$ 的路徑總和。

📝 主範例 6.6　遲到的訂單來自哪裡？（延續主範例 6.4）

情境：客服接到一筆遲到的客訴訂單。在不看地址的情況下，這筆訂單來自郊區的機率是多少？

用樹狀圖的結果直接組裝：

$$P(D^C \mid L) = \frac{P(D^C \cap L)}{P(L)} = \frac{0.09}{0.16} = 0.5625$$

判讀：先驗上郊區只占 30%，但「遲到」這個證據把它修正到 56%——證據讓機率翻了近一倍。

✅ 貝氏定理的本質就是條件機率＋全機率法則，沒有新的數學——新的是思考方向：從「原因 → 結果」翻轉成「結果 → 原因」。

📝 主範例 6.7　篩檢陽性，該多恐慌？

情境：某疾病在無症狀成人中的盛行率為 1%。快篩試劑的性能：真有病時呈陽性的機率（靈敏度）為 98%；沒病時誤呈陽性的機率（偽陽性率）為 2%。阿哲快篩呈陽性，他真的有病的機率是多少？

Step 1　列出資訊：$P(D) = 0.01$、$P(+ \mid D) = 0.98$、$P(+ \mid D^C) = 0.02$。

Step 2　全機率法則算 $P(+)$：

$$P(+) = (0.01)(0.98) + (0.99)(0.02) = 0.0098 + 0.0198 = 0.0296$$

Step 3　貝氏定理：

$$P(D \mid +) = \frac{0.0098}{0.0296} \approx 0.331$$

判讀：試劑「98% 準確」，但陽性者真正有病的機率只有 33%！

直覺化：想像 10,000 人受檢——有病的 100 人中約 98 人驗出陽性；沒病的 9,900 人中約 198 人誤判陽性。陽性總共 296 人，其中真有病的只有 98 人（33%）。因為沒病的人基數太大，偽陽性的「絕對人數」反而壓過真陽性。

⚠️ 這就是「基本率謬誤 (base rate neglect)」——忽略先驗機率（盛行率）的直覺錯誤。它解釋了為什麼醫師會要求陽性者「複檢」而不是直接開始治療，也是罕見疾病不宜全面普篩的統計學理由。

📚 綜合練習 (Comprehensive Practice)

以下習題涵蓋機率指派、聯合／條件機率、機率法則與貝氏定理，請先獨立作答，再展開解答確認。

I. 概念複習

  1. 樣本空間 $S$ 有 25 個機會均等的結果，事件 $A$ 包含 10 個結果。求 $P(A)$ 與 $P(A^C)$。

    ▶ 點此展開解答

    $P(A) = 10/25 = 0.4$；$P(A^C) = 1 - 0.4 = 0.6$。

  2. 已知 $P(A) = 0.55$、$P(B) = 0.35$、$P(A \cup B) = 0.75$。求 $P(A \cap B)$，並判斷 $A$、$B$ 是否互斥、是否獨立。

    ▶ 點此展開解答

    由加法法則移項：$P(A \cap B) = 0.55 + 0.35 - 0.75 = 0.15$。

    互斥？$P(A \cap B) = 0.15 \neq 0$ → 不互斥。

    獨立？$P(A)P(B) = 0.55 \times 0.35 = 0.1925 \neq 0.15$ → 不獨立。

  3. 解釋「互斥」與「獨立」的差異，並說明為何兩個機率皆為正的互斥事件必然不獨立。

    ▶ 點此展開解答

    互斥描述「能否同時發生」（$P(A \cap B) = 0$）；獨立描述「資訊是否互相影響」（$P(A \mid B) = P(A)$）。

    若 $A$、$B$ 互斥且 $P(A), P(B) > 0$：知道 $B$ 發生後，$A$ 必不可能發生，$P(A \mid B) = 0 \neq P(A) > 0$——資訊影響巨大，故必不獨立。

II. 計算與應用

  1. 6.1　戳戳樂的機率指派

    夜市戳戳樂共 40 格：大獎 4 格、中獎 10 格、小獎 26 格。每格被戳的機會均等，戳一格：

    • a. 寫出這個隨機試驗的樣本空間（以獎項分類）。
    • b. 求戳中大獎、中獎、小獎的機率，並驗證總和為 1。
    • c. 求「戳中大獎或中獎」的機率。
    ▶ 點此展開解答

    a. $S = \{\text{大獎、中獎、小獎}\}$（互斥且周延）。

    b. $P(\text{大}) = 4/40 = 0.10$；$P(\text{中}) = 10/40 = 0.25$；$P(\text{小}) = 26/40 = 0.65$。總和 $= 1$ ✓

    c. 大獎與中獎互斥：$P(\text{大} \cup \text{中}) = 0.10 + 0.25 = 0.35$。

  2. 6.2　打工與延畢的聯合機率表

    某校追蹤畢業生資料，整理出「在學期間每週打工超過 20 小時（$W$）」與「延畢（$Y$）」的聯合機率表：

    延畢 $Y$如期畢業 $Y^C$
    高工時打工 $W$0.090.21
    非高工時 $W^C$0.070.63
    • a. 求 $P(W)$ 與 $P(Y)$（邊際機率）。
    • b. 求 $P(Y \mid W)$ 與 $P(Y \mid W^C)$，並比較。
    • c. 高工時打工與延畢是否獨立？
    • d. 「所以打工害人延畢」——這個結論妥當嗎？
    ▶ 點此展開解答

    a. $P(W) = 0.09 + 0.21 = 0.30$；$P(Y) = 0.09 + 0.07 = 0.16$。

    b. $P(Y \mid W) = 0.09/0.30 = 0.30$；$P(Y \mid W^C) = 0.07/0.70 = 0.10$。高工時者的延畢率是其他人的 3 倍。

    c. $P(Y \mid W) = 0.30 \neq P(Y) = 0.16$ → 不獨立（相關）。

    d. 不妥。這是觀察資料——「經濟壓力」可能同時導致高工時打工與延畢（混雜因素）。相關支持進一步調查（如提供助學金後追蹤），但不能直接下因果結論。

  3. 6.3　刮刮卡不放回抽取

    咖啡店促銷活動的抽獎箱有 10 張刮刮卡，其中 3 張中獎。小樂連抽 2 張（不放回）：

    • a. 兩張都中獎的機率？
    • b. 恰好一張中獎的機率？
    • c. 至少一張中獎的機率？（用補集法則驗算）
    ▶ 點此展開解答

    a. $P(\text{中}_1 \cap \text{中}_2) = \frac{3}{10} \times \frac{2}{9} = \frac{6}{90} \approx 0.067$

    b. 兩條路徑：中→未 $= \frac{3}{10} \times \frac{7}{9} = \frac{21}{90}$；未→中 $= \frac{7}{10} \times \frac{3}{9} = \frac{21}{90}$。合計 $\frac{42}{90} \approx 0.467$。

    c. $P(\text{至少一張}) = 1 - P(\text{都沒中}) = 1 - \frac{7}{10} \times \frac{6}{9} = 1 - \frac{42}{90} = \frac{48}{90} \approx 0.533$。
    驗算：$\frac{6}{90} + \frac{42}{90} = \frac{48}{90}$ ✓（a + b = c）

    不放回 → 第二抽的機率依賴第一抽的結果（條件機率）；若「抽後放回」則兩抽獨立，機率變為 $0.3 \times 0.3 = 0.09$。

  4. 6.4　雙重驗證的防護力

    某帳號有兩道獨立防線：密碼被破解的機率 0.05、手機驗證被攻破的機率 0.02。

    • a. 帳號被完全入侵（兩道都被攻破）的機率？
    • b. 至少一道防線被攻破的機率？
    • c. 若兩道防線其實「不獨立」（例如手機裡存了密碼備忘錄），(a) 的答案會如何變化？
    ▶ 點此展開解答

    a. 獨立 → $P = 0.05 \times 0.02 = 0.001$（千分之一）。

    b. $1 - (0.95)(0.98) = 1 - 0.931 = 0.069$。

    c. 若手機被攻破時密碼幾乎必然一起洩漏，$P(\text{密碼破} \mid \text{手機破}) \gg 0.05$，聯合機率將遠大於 0.001（最壞情況接近 0.02）。「多重防護」的價值建立在獨立性上——把雞蛋放在會一起破的籃子裡，等於只有一個籃子。

  5. 6.5　兩條產線的品質追溯（貝氏定理）

    某工廠的行動電源由兩條產線生產：A 線佔產量 60%（不良率 2%）、B 線佔 40%（不良率 5%）。品管在出貨區隨機抽到一顆不良品：

    • a. 畫樹狀圖，求整體不良率 $P(F)$。
    • b. 這顆不良品來自 B 線的機率？
    • c. 廠長說「B 線只占四成產量，不良品應該多半來自 A 線」。請評論。
    ▶ 點此展開解答

    a. 聯合機率：A 且不良 $= 0.6 \times 0.02 = 0.012$；B 且不良 $= 0.4 \times 0.05 = 0.020$。

    $$P(F) = 0.012 + 0.020 = 0.032$$

    b.

    $$P(B \mid F) = \frac{0.020}{0.032} = 0.625$$

    c. 廠長犯了忽略「不良率差異」的錯：B 線雖然產量較少，但不良率是 A 線的 2.5 倍，因此不良品中有 62.5% 來自 B 線。改善品質應優先督導 B 線。

  6. 6.6　垃圾郵件過濾器（貝氏定理）

    已知 25% 的來信是垃圾郵件。過濾器對垃圾郵件的攔截率（判為垃圾）是 92%；對正常郵件的誤判率是 4%。

    • a. 隨機一封信被判為垃圾郵件的機率？
    • b. 一封被判為垃圾的信，真的是垃圾郵件的機率？
    • c. 一封被放行的信，其實是垃圾郵件的機率？
    ▶ 點此展開解答

    a. $P(\text{判垃圾}) = (0.25)(0.92) + (0.75)(0.04) = 0.23 + 0.03 = 0.26$

    b. $P(\text{垃圾} \mid \text{判垃圾}) = \dfrac{0.23}{0.26} \approx 0.885$——攔下來的信約 88.5% 真的是垃圾。

    c. 被放行 $= 1 - 0.26 = 0.74$；其中漏網的垃圾 $= (0.25)(0.08) = 0.02$。$P(\text{垃圾} \mid \text{放行}) = 0.02/0.74 \approx 0.027$——收件匣裡約 2.7% 仍是漏網垃圾。

📌 Chapter Summary

  • 基礎：樣本空間互斥且周延；每個結果 $0 \leq P \leq 1$、總和 $= 1$。指派方法：古典（對稱機制）、相對頻率（歷史資料）、主觀（一次性事件）。
  • 三種機率：聯合 $P(A \cap B)$、邊際（表格整列/行加總）、條件 $P(A \mid B) = P(A \cap B)/P(B)$——「把世界縮小到 B 再看 A 的占比」。
  • 獨立：$P(A \mid B) = P(A)$，等價於 $P(A \cap B) = P(A)P(B)$。互斥 ≠ 獨立——互斥（且機率為正）的事件必不獨立。
  • 三大法則：補集（至少一次 = 1 − 全落空）、加法（減交集防重複）、乘法（$P(A)P(B \mid A)$，樹狀圖沿路徑相乘）。全機率法則：把各分支的聯合機率加總。
  • 貝氏定理：後驗 $= \dfrac{\text{目標路徑}}{\text{所有含證據的路徑總和}}$——從結果回推原因。先驗（基本率）很低時，即使檢測很準，後驗仍可能出乎意料地低（基本率謬誤）。
