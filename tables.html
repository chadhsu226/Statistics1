Z Z 分配（Standard Normal Distribution）
定義：標準常態分配是均值為 0、標準差為 1 的常態分配，記作 Z ~ N(0, 1)。
f(z) = (1/√(2π)) · e^(−z²/2)，　−∞ < z < +∞
任何常態隨機變數 X ~ N(μ, σ²) 經標準化後得到 Z = (X − μ) / σ ~ N(0, 1)。

t t 分配（Student's t-Distribution）
定義：若 Z ~ N(0,1)，V ~ χ²(ν) 且兩者獨立，則
t = Z / √(V/ν) ~ t(ν)
實務上，t 統計量由 t = (X̄ − μ₀) / (s/√n) 計算，自由度 ν = n − 1。

為什麼要用 t 分配而非 Z 分配？

使用樣本標準差 s 估計未知的母體標準差 σ 時，引入了額外的估計誤差。t 統計量的分母包含兩個隨機變數（X̄ 和 s），造成尾部比 Z 分配更厚。自由度 ν = n−1 越大，s 越穩定，t 分配越接近 Z 分配。當 ν ≥ 120 時，t 分配已非常接近 Z 分配，實務上可直接用 Z 臨界值。

χ² 卡方分配（Chi-Square Distribution）
定義：若 Z₁, Z₂, …, Zᵥ 為 ν 個獨立的標準常態隨機變數，則其平方和
χ²(ν) = Z₁² + Z₂² + … + Zᵥ²
服從自由度 ν 的卡方分配。

常見應用場景

① 推論母體變異數：(n−1)S²/σ₀² ~ χ²(n−1)，用於假設檢定與信賴區間。

② 適合度檢定（Goodness-of-fit）：觀測次數是否符合理論分配。

③ 獨立性檢定（列聯表）：判斷兩個類別變數是否獨立。

④ 同質性檢定：比較多個母體的比例是否相同。

F F 分配（F Distribution）
定義：若 χ²(d₁) 與 χ²(d₂) 為兩個獨立的卡方隨機變數，則
F = [χ²(d₁)/d₁] / [χ²(d₂)/d₂] ~ F(d₁, d₂)
實務上由 F = s₁²/s₂² 計算，其中 d₁ = n₁−1，d₂ = n₂−1。

基本性質

定義域
F ≥ 0

常見應用場景

① 比較兩母體變異數（§13.4）：判斷兩組資料的波動是否有顯著差異；亦是進行 Pooled t 檢定前的前置 F 檢定。

② ANOVA（變異數分析）（§14）：F = 組間均方 / 組內均方，檢定多組均值是否相等。

③ 迴歸分析顯著性（§16）：F 統計量用於檢定整體迴歸模型的顯著性。
