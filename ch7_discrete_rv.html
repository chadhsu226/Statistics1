Chapter 7　隨機變數與離散機率分配 (Random Variables and Discrete Probability Distributions)

§7.1　隨機變數與離散機率分配 (Random Variables and Probability Distributions)

一、隨機變數

📐 定義

隨機變數 (Random Variable, $X$)：把隨機試驗的每一個結果對應到一個數值的函數。

  • 離散 (Discrete)：可能值是有限個或可數無限個。例：每小時的訂單數、10 次罰球的命中數。
  • 連續 (Continuous)：可能值布滿一段區間、不可數。例：等待時間、重量、報酬率（→ 第 8 章）。

判斷心法：可以「一個一個數」的是離散；只能「量出來」的是連續。

二、離散機率分配（PMF）

📐 機率分配的兩個必要條件

機率分配：列出 $X$ 的每個可能值 $x$ 及其機率 $P(x)$。合法的 PMF 必須滿足：

$$0 \leq P(x) \leq 1 \ \text{對所有 } x \qquad \text{且} \qquad \sum_{x} P(x) = 1$$

三、期望值、變異數與標準差

📐 母體參數

$$\mu = E(X) = \sum x\,P(x)$$
$$\sigma^2 = V(X) = \sum (x - \mu)^2 P(x) \;=\; \sum x^2 P(x) - \mu^2 \qquad \sigma = \sqrt{\sigma^2}$$

期望值是「長期平均」——重複這個隨機試驗無限多次，結果的平均值。它不必是 $X$ 可能出現的值（平均每戶 1.3 個小孩，沒有任何一戶真的有 1.3 個）。捷徑公式 $\sum x^2P(x) - \mu^2$ 通常比定義式好算。

📝 主範例 7.1　手搖飲店的每小時線上訂單

情境：「沐茶」以歷史資料整理出尖峰時段每小時線上訂單數 $X$ 的機率分配：
$x$（單）01234
$P(x)$0.100.200.400.200.10

a. 驗證這是合法的機率分配：每個 $P(x) \in [0,1]$ ✓；總和 $= 0.10+0.20+0.40+0.20+0.10 = 1$ ✓

b. 求 $P(X \geq 3)$ 與 $P(1 \leq X \leq 3)$：

$$P(X \geq 3) = 0.20 + 0.10 = 0.30 \qquad P(1 \leq X \leq 3) = 0.20 + 0.40 + 0.20 = 0.80$$

c. 期望值：

$$\mu = 0(0.1) + 1(0.2) + 2(0.4) + 3(0.2) + 4(0.1) = 2.0 \text{ 單／小時}$$

d. 變異數與標準差（捷徑公式）：

$$\sum x^2 P(x) = 0 + 0.2 + 1.6 + 1.8 + 1.6 = 5.2 \qquad \sigma^2 = 5.2 - 2^2 = 1.2 \qquad \sigma = 1.095$$
✅ 判讀：長期而言平均每小時 2 單、標準差約 1.1 單。店長可據此配置外送人力——例如以「$\mu + 2\sigma \approx 4.2$ 單」作為尖峰備援的規劃上限。

§7.2　期望值與變異數的運算法則 (Laws of Expected Value and Variance)

📐 線性轉換 $Y = aX + b$ 的法則

$$E(aX + b) = a\,E(X) + b \qquad\qquad V(aX + b) = a^2\,V(X) \qquad\qquad \sigma_{aX+b} = |a|\,\sigma_X$$

兩個直覺：① 常數 $b$ 把整個分布「平移」，會改變平均、不改變離散程度（所以 $V$ 不含 $b$）；② 尺度 $a$ 同時放大平均與離散，變異數是「平方尺度」所以乘 $a^2$。

📝 範例 7.2　從訂單數到時薪利潤（延續主範例 7.1）

情境：沐茶每筆線上訂單平均賺 $40，該時段的固定成本（人力＋系統）為每小時 $50。設每小時利潤 $Y = 40X - 50$。求 $Y$ 的期望值與標準差。
$$E(Y) = 40\,E(X) - 50 = 40(2.0) - 50 = 30 \text{ 元／小時}$$
$$\sigma_Y = |40| \times \sigma_X = 40 \times 1.095 = 43.8 \text{ 元}\qquad V(Y) = 40^2 \times 1.2 = 1920$$
✅ 平均每小時賺 $30，但標準差高達 $43.8——虧錢的小時並不罕見（$X \leq 1$ 時 $Y \leq -10$，機率 0.30）。期望值為正不代表「每小時都賺」，離散程度提醒了風險。

§7.3　二項分配 (Binomial Distribution)

一、二項試驗的四個條件

📐 二項試驗 (Binomial Experiment)

  1. 固定次數：試驗重複 $n$ 次。
  2. 兩種結果：每次試驗只分「成功／失敗」。
  3. 機率恆定：每次成功機率都是 $p$。
  4. 相互獨立：各次試驗互不影響。

$X$ ＝ $n$ 次試驗中的成功次數，記作 $X \sim B(n, p)$。
常見的「不獨立」陷阱：從小母體不放回抽樣（抽到的會改變剩下的比例）。母體遠大於樣本（$N \gg n$，例如民調）時，違反程度可忽略，仍可用二項分配近似。

二、二項機率公式與參數

📐 PMF、平均數與變異數

$$P(X = x) = \binom{n}{x} p^x (1-p)^{n-x}, \qquad x = 0, 1, \ldots, n \qquad \left(\binom{n}{x} = \frac{n!}{x!\,(n-x)!}\right)$$
$$\mu = np \qquad \sigma^2 = np(1-p) \qquad \sigma = \sqrt{np(1-p)}$$

公式三部件的直覺：$p^x$（指定的 $x$ 次成功）× $(1-p)^{n-x}$（其餘全失敗）× $\binom{n}{x}$（成功可以出現在哪幾次的排列數）。

📝 主範例 7.3　準時到貨保證

情境：某披薩店推出「30 分鐘到貨」保證，歷史準時率為 90%，且各訂單相互獨立。今晚接到 5 筆外送訂單，令 $X$ ＝ 準時送達的訂單數。

a. 確認二項條件：$n = 5$ 固定 ✓；準時／遲到兩種結果 ✓；$p = 0.9$ 恆定 ✓；訂單獨立 ✓ → $X \sim B(5, 0.9)$。

b. 全部準時的機率：

$$P(X = 5) = \binom{5}{5}(0.9)^5(0.1)^0 = 0.5905$$

c. 至少 4 筆準時：

$$P(X \geq 4) = P(4) + P(5) = \binom{5}{4}(0.9)^4(0.1)^1 + 0.5905 = 0.3281 + 0.5905 = 0.9185$$

d. 「至少一筆遲到」的機率（補集技巧）：

$$P(\text{至少一筆遲到}) = 1 - P(X = 5) = 1 - 0.5905 = 0.4095$$

e. 平均數與標準差：

$$\mu = np = 5(0.9) = 4.5 \qquad \sigma = \sqrt{5(0.9)(0.1)} = \sqrt{0.45} = 0.671$$
✅ 單筆準時率高達九成，但只要接 5 單，「至少一單開天窗」的機率就有四成——這是「壞事至少發生一次」隨規模放大的必然結果，也是為什麼服務水準協議（SLA）的成本會隨訂單量暴增。

📝 範例 7.4　罰球十投

情境：校隊球員罰球命中率 70%，各投獨立。罰 10 球，令 $X$ ＝ 命中數 $\sim B(10, 0.7)$。

a. 恰好命中 7 球：

$$P(X = 7) = \binom{10}{7}(0.7)^7(0.3)^3 = 120 \times 0.08235 \times 0.027 = 0.2668$$

b. 命中 8 球以上：

$$P(X \geq 8) = P(8) + P(9) + P(10) = 0.2335 + 0.1211 + 0.0282 = 0.3828$$

c. 平均命中 $\mu = 10(0.7) = 7$ 球，$\sigma = \sqrt{10(0.7)(0.3)} = \sqrt{2.1} = 1.449$ 球。

✅ 注意「平均 7 球」但「恰好 7 球」的機率只有 0.27——期望值是分布的重心，不是最可能發生就一定發生的保證。完整的分布形狀請用文末的互動計算機觀察。

§7.4　卜瓦松分配 選讀補充

📐 卜瓦松分配 (Poisson Distribution)

描述「一段固定時間或空間內，某事件發生次數」的分配（沒有固定的試驗次數 $n$，事件在連續的時間軸上隨機出現）。設單位期間的平均發生次數為 $\mu$：

$$P(X = x) = \frac{e^{-\mu}\,\mu^x}{x!}, \qquad x = 0, 1, 2, \ldots \qquad E(X) = V(X) = \mu$$

典型場景：每小時的客服來電數、每頁書稿的錯字數、每天的機台故障數。與二項的分野：二項問「$n$ 次中成功幾次」，卜瓦松問「這段期間內發生幾次」。

📝 範例 7.5　客服中心的深夜來電

情境：某銀行客服深夜時段平均每小時接到 $\mu = 3$ 通來電，來電次數服從卜瓦松分配。

a. 一小時內完全沒有來電：$P(X = 0) = e^{-3} = 0.0498$

b. 一小時內來電 4 通以上：

$$P(X \geq 4) = 1 - \left[P(0)+P(1)+P(2)+P(3)\right] = 1 - e^{-3}\left(1 + 3 + 4.5 + 4.5\right) = 1 - 0.6472 = 0.3528$$
✅ 深夜只排一位客服（假設每小時最多能處理 3 通），約有 35% 的小時會爆線——排班決策立刻有了量化依據。

📚 綜合練習 (Comprehensive Practice)

以下習題涵蓋 PMF 機率計算、期望值與變異數、線性轉換與二項分配，請先獨立作答，再展開解答確認。

I. 概念複習

  1. 離散機率分配必須滿足哪兩個條件？隨機變數「離散」與「連續」如何區分？

    ▶ 點此展開解答

    兩條件：① $0 \leq P(x) \leq 1$；② $\sum P(x) = 1$。

    離散：可能值可一一列舉（計數型）；連續：可能值布滿區間、不可數（度量型）。

  2. 二項試驗的四個條件是什麼？各舉一個「違反該條件」的例子。

    ▶ 點此展開解答
    1. 固定次數——違例：投籃「直到進 3 球為止」（次數不固定）。
    2. 兩種結果——違例：骰子點數（六種結果；除非重新定義為「是否為 6 點」）。
    3. 機率恆定——違例：射飛鏢越玩越熟練（p 隨試驗改變）。
    4. 獨立性——違例：小母體不放回抽樣（前次結果改變後次機率）。

  3. 為什麼 $V(X + b) = V(X)$（加常數不改變變異數），而 $V(aX) = a^2 V(X)$？請用直覺說明。

    ▶ 點此展開解答

    加常數 $b$ 是把整個分布原封不動地平移——每個值與平均數的距離完全不變，所以離散程度不變。

    乘上 $a$ 則把每個「離差」也放大 $a$ 倍；變異數是離差的平方平均，故放大 $a^2$ 倍（標準差放大 $|a|$ 倍）。

II. 計算與應用

  1. 7.1　自助洗衣店的機台使用

    某自助洗衣店整理出尖峰時段「使用中洗衣機台數 $X$」的機率分配：

    $x$012345
    $P(x)$0.050.150.250.300.150.10
    • a. 求 $P(2 \leq X \leq 4)$、$P(X > 4)$、$P(X < 2)$。
    • b. 計算 $\mu$。
    • c. 計算 $\sigma$。
    ▶ 點此展開解答

    a. $P(2 \leq X \leq 4) = 0.25+0.30+0.15 = 0.70$；$P(X > 4) = 0.10$；$P(X < 2) = 0.05+0.15 = 0.20$。

    b. $\mu = 0(0.05)+1(0.15)+2(0.25)+3(0.30)+4(0.15)+5(0.10) = 0.15+0.50+0.90+0.60+0.50 = 2.65$ 台。

    c. $\sum x^2P(x) = 0.15+1.00+2.70+2.40+2.50 = 8.75$；$\sigma^2 = 8.75 - 2.65^2 = 8.75 - 7.0225 = 1.7275$；$\sigma = 1.314$ 台。

  2. 7.2　用電量的線性轉換（承 7.1）

    每台使用中的洗衣機每小時耗電 1.2 度，店內基礎照明另固定耗電 0.8 度。設每小時總用電 $Y = 1.2X + 0.8$。

    • a. 求 $E(Y)$。
    • b. 求 $\sigma_Y$。
    ▶ 點此展開解答

    a. $E(Y) = 1.2(2.65) + 0.8 = 3.18 + 0.8 = 3.98$ 度。

    b. $\sigma_Y = 1.2 \times 1.314 = 1.577$ 度。（常數 0.8 不影響標準差。）

  3. 7.3　亂猜兩題是非題

    小考最後兩題是非題完全不會，決定亂猜（每題答對機率 0.5，互相獨立）。令 $X$ ＝ 答對題數。

    • a. 用樹狀圖列出四種結果與機率，寫出 $X$ 的機率分配。
    • b. 求 $P(X \geq 1)$。
    • c. 驗證 $\mu = np$、$\sigma^2 = np(1-p)$ 與定義式計算的結果一致。
    ▶ 點此展開解答

    a. 四條路徑（對對、對錯、錯對、錯錯）各 0.25。分配：$P(0)=0.25$、$P(1)=0.50$、$P(2)=0.25$。

    b. $P(X \geq 1) = 1 - P(0) = 0.75$。

    c. 定義式：$\mu = 0(0.25)+1(0.5)+2(0.25) = 1$；$\sum x^2P = 0+0.5+1.0 = 1.5$，$\sigma^2 = 1.5 - 1 = 0.5$。
    公式：$\mu = np = 2(0.5) = 1$ ✓；$\sigma^2 = 2(0.5)(0.5) = 0.5$ ✓——兩者一致（是非題亂猜正是 $B(2, 0.5)$）。

  4. 7.4　二項公式練習

    $X \sim B(12,\ 0.2)$。用公式計算：

    • a. $P(X = 3)$
    • b. $P(X = 0)$
    • c. $P(X \geq 1)$
    ▶ 點此展開解答

    a. $P(3) = \binom{12}{3}(0.2)^3(0.8)^9 = 220 \times 0.008 \times 0.13422 = 0.2362$

    b. $P(0) = (0.8)^{12} = 0.0687$

    c. $P(X \geq 1) = 1 - 0.0687 = 0.9313$

  5. 7.5　快篩試劑的當日銷售分配

    藥局每天上架 4 盒某快篩組合包，依歷史資料每盒當日售出的機率為 0.6，各盒獨立。令 $X$ ＝ 當日售出盒數，寫出 $X$ 的完整機率分配，並求 $\mu$ 與 $\sigma^2$。

    ▶ 點此展開解答

    $X \sim B(4, 0.6)$：

    $x$01234
    $P(x)$0.02560.15360.34560.34560.1296

    計算例：$P(2) = \binom{4}{2}(0.6)^2(0.4)^2 = 6 \times 0.36 \times 0.16 = 0.3456$。總和 $= 1$ ✓

    $\mu = np = 2.4$ 盒；$\sigma^2 = np(1-p) = 4(0.6)(0.4) = 0.96$。

  6. 7.6　電話行銷的期望成果

    客服中心每通外撥電話成功推銷加值方案的機率為 0.15，一位客服每天撥 30 通（各通獨立）。

    • a. 每天成功數的平均與標準差？
    • b. 「一整天掛零」的機率？
    • c. 主管要求每天至少成功 1 件，這個要求合理嗎？
    ▶ 點此展開解答

    a. $\mu = 30(0.15) = 4.5$ 件；$\sigma = \sqrt{30(0.15)(0.85)} = \sqrt{3.825} = 1.956$ 件。

    b. $P(X = 0) = (0.85)^{30} = 0.0076$。

    c. $P(X \geq 1) = 1 - 0.0076 = 0.9924$——正常發揮下 99% 以上的天數都能達標，要求合理；連續多天掛零反而是「$p$ 已改變」的警訊（名單品質、話術問題），值得管理者調查。

📌 Chapter Summary

  • 隨機變數把試驗結果數值化：離散（可數）vs 連續（不可數）。合法 PMF：各機率 $\in [0,1]$ 且總和 $= 1$。
  • 期望值 $\mu = \sum xP(x)$ 是長期平均；變異數用捷徑 $\sigma^2 = \sum x^2P(x) - \mu^2$。
  • 線性轉換：$E(aX+b) = aE(X)+b$；$V(aX+b) = a^2V(X)$——平移不改變離散，尺度以平方放大變異數。
  • 二項分配 $B(n,p)$：固定次數、兩種結果、機率恆定、獨立。$P(x) = \binom{n}{x}p^x(1-p)^{n-x}$；$\mu = np$、$\sigma^2 = np(1-p)$。「至少一次」用補集。
  • 卜瓦松分配（選讀）：固定期間內事件發生次數，$P(x) = e^{-\mu}\mu^x/x!$，期望值 = 變異數 = $\mu$。
